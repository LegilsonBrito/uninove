##### Comentário de apenas uma linha com ```//```:
```php
<?php
  // Utilizando o símbolo de barra duas vezes para comentar uma linha de código
?>
```

##### Comentário de apenas uma linha com ```#```:
```php
<?php     
  # Utilizando o símbolo de barra duas vezes para comentar uma linha de código
?>
```
##### Comentário de várias linhas com ```/* */```:
```php
<?php     
  /* Este é um comentário de múltiplas      
  de comentários */
?>
```
